# Auto-Assigner

Automatically distribute unassigned tickets across the available members of an
assignment group, fairly and on a schedule — with shifts, breaks, working
hours, and round-robin balancing.

- **[User Guide](user-guide.md)** — how to set up and run an auto-assigner.
